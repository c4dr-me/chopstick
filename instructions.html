Chopstick Game Instructions

Game Setup

  • Both Player 1 and Player 2 start with 1 finger on each hand.

Hand Selection

  • A player can select their own hand if it has 4 or fewer fingers. The selected hand will be highlighted.
  • A player can attack an opponent's hand with their selected hand.

Attack

  • When a hand is attacked, the number of fingers on the attacking hand is added to the attacked hand.
  • If the total number of fingers on the attacked hand exceeds 4, it becomes a "fist" (represented as 5 fingers).

Splitting

  • A player can split the total number of fingers between their two hands if the total number of fingers is even and greater than or equal to 4.
  • The split distributes the total fingers equally between the two hands.
  • Splitting cannot occur if both hands already have 4 fingers each (maximum allowed).

Turn Management

  • After an attack or split, the turn ends and switches to the other player.
  • The game checks if both hands of a player exceed 4 fingers, which would mean the other player wins.

Game Over and Reset

  • If a player has both hands with more than 4 fingers, the game declares the other player as the winner.
  • The game can be reset to the initial state.

Hand Display

  • The number of fingers on each hand is displayed using corresponding SVG images.
  • The images are updated dynamically based on the game state.
